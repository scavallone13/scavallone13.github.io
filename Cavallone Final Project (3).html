Project Goals

For my final project, I am exploring the correlation between campaign donations, spending and government election victories with a focus on presidential elections. The data I will be looking at is from federal elections from the last 24 years. I am setting out to provide evidence that whoever spends more money in an elections campaign will win the election. I will be using supplementary data from congressional elections to display this winning trend on a greater sample. The secondary goal, and also application of this analysis would be to predict presidential and congressional elections. The inspiration for this project comes from the upcoming 2024 election, and hopefully my model would be able to predict the breakdowns of the different candidates victories at varying levels of government.

Data Set 1

The first data set I will be working with comes from the Federal Election Commission (FEC). This data set contains the summary of financial data for each candidate that raised or spent money during the 2023-2024 period. The data is available and summarized regardless of if the candidate is up for election within this cycle.

The file shows one record per candidate and has the following information about the candidate:

total receipts, transfers received from authorized committees, total disbursements, transfers given to authorized committees, cash-on-hand totals, loans and debts, and other financial summary information All Candidates File Description (FEC).

I am working with this data set because it provides the most recent data on the candidates funding statuses at this point in time. The Federal Election Commission also provides the most accurate spending and raising reports for all candidates as provided by the candidates and all associated parties.

Important Observations

At this point, there are a few things we can identify about the data: The COH_BOP and the COH_COP values tell a very interesting story about the fundraising and spending habits of both candidate's campaigns.

  1. The COH_BOP value represents the funds that both candidates had at the beginning of the 2023 period. As expected, the Trump campaign had more money than the Harris campaign because Harris was not yet running for president. However, this data suggests that there was already funding in Harris' campaign, most likely from her campaign with Biden. We will need to further investigate to clarify this.
  2. The COH_COP value represents the funds that both candidates had at the most recent coverage end date, which was August 31, 2024, according to the table above. As we can see there, the Harris campaign's finances climbed drastically, ending with a sum greater than the Trump campaign's.
  3. The TTL_INDIV_CONTRIB value represents the total individual contributions by candidate. As we can see here, the Harris campaign has a significantly greater sum of contributions than the Trump campaign.

import pandas as pd

# Assuming your DataFrame is named 'df'
def assign_winner(df):
  """
  Assigns a value of 1 to the 'winner' column based on the year and candidate.

  Args:
      df: The DataFrame to modify.
  """
  df['winner'] = 0  # Initialize all values in 'winner' column to 0

# Set winner to 1 for the rest of the presidents

  df.loc[(df['year'] == 2000) & (df['candidate'] == 'BUSH, GEORGE W.'), 'winner'] = 1
  df.loc[(df['year'] == 2004) & (df['candidate'] == 'BUSH, GEORGE W.'), 'winner'] = 1
  df.loc[(df['year'] == 2008) & (df['candidate'] == 'OBAMA, BARACK H.'), 'winner'] = 1
  df.loc[(df['year'] == 2012) & (df['candidate'] == 'OBAMA, BARACK H.'), 'winner'] = 1
  df.loc[(df['year'] == 2016) & (df['candidate'] == 'TRUMP, DONALD J.'), 'winner'] = 1
  df.loc[(df['year'] == 2020) & (df['candidate'] == 'BIDEN, JOSEPH R. JR'), 'winner'] = 1
  return df

import pandas as pd
import matplotlib.pyplot as plt

def plot_winning_parties(df):
    """
    Creates a line graph showing the parties of presidential election winners,
    with 'REP' in red and 'DEM' in blue, applying the logic to all years.

    Args:
        df: The DataFrame containing election data (presidential_winners).
    """

    # Create a copy of the DataFrame to avoid modifying the original
    plot_df = df.copy()

    # Apply the logic for all years for both parties
    for year in plot_df['year'].unique():
        # Use .values[0] instead of .iloc[0] to handle empty Series
        # and add error handling for years without a winner
        try:
            winner_party = plot_df.loc[(plot_df['year'] == year) & (plot_df['winner'] == 1), 'party_simplified'].values[0]
        except IndexError:
            # Handle cases where there's no winner for the year
            print(f"Warning: No winner found for year {year}. Skipping...")
            continue  # Skip to the next year

        plot_df.loc[(plot_df['year'] == year) & (plot_df['party_simplified'] != winner_party), 'winner'] = 0
        plot_df.loc[(plot_df['year'] == year) & (plot_df['party_simplified'] == winner_party), 'winner'] = 1

    # Create the line plot with separate lines for each party
    plt.figure(figsize=(10, 6))  # Adjust figure size if needed

    for party, color in zip(['REPUBLICAN', 'DEMOCRAT'], ['red', 'blue']):
        party_data = plot_df[plot_df['party_simplified'] == party]
        plt.plot(party_data['year'], party_data['winner'], marker='o', linestyle='-', label=party, color=color)

    # Customize the plot
    plt.title("Winning Parties in Presidential Elections")
    plt.xlabel("Year")
    plt.ylabel("Winner (1 = Won)")
    plt.yticks([0, 1])  # Set y-axis ticks to 0 and 1
    plt.xticks(plot_df['year'].unique())  # Set x-axis ticks to election years
    plt.grid(True)  # Add grid lines
    plt.legend()  # Add a legend to show party colors

    plt.show()
